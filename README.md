
# 新东方亚英案例查询系统

3万+留学申请案例数据，支持多条件筛选和快速检索的在线查询系统。

## 功能特点

- 多条件筛选：学生姓名、本科学校、专业、学校层次、目标学校等
- 不区分大小写搜索
- 模糊匹配查询
- 分页显示，每页15条数据
- 点击学生姓名查看详细信息
- 实时统计显示

## 部署到 GitHub Pages

### 方法一：通过 GitHub Actions 自动部署（推荐）

1. 将代码推送到 GitHub 仓库
2. 进入仓库的 **Settings** → **Pages**
3. 在 **Source** 下选择 **GitHub Actions**
4. 推送代码到 `main` 或 `master` 分支，会自动触发部署

### 方法二：手动部署

1. 进入仓库的 **Settings** → **Pages**
2. 在 **Source** 下选择 **Deploy from a branch**
3. 选择 `main` 或 `master` 分支，选择 `/ (root)` 目录
4. 点击 **Save**

## 多用户访问说明

本系统是纯前端静态页面，部署到 GitHub Pages 后：

- **完全支持多用户同时访问**
- **每个用户的搜索和筛选互不影响**
- **无需后端服务器**
- **无需数据库**

所有数据文件（*.js）在用户浏览器端加载，每个用户独立进行搜索和筛选操作。

## 使用说明

1. 在搜索框中输入关键词（支持大小写不敏感和模糊匹配）
2. 点击"搜索"按钮或按回车键
3. 点击学生姓名查看详细信息
4. 使用分页按钮浏览更多数据
5. 点击"重置"按钮清空搜索条件

## 浏览器兼容性

- Chrome (推荐)
- Firefox
- Safari
- Edge
- 其他现代浏览器

## 性能优化

- 使用 `.nojekyll` 文件禁用 Jekyll 处理，提高加载速度
- 数据文件按需加载
- 分页显示减少 DOM 操作
